## Problem
Needed a reliable way to aggregate pricing data from multiple competitor websites without getting blocked.

## Approach
Developed a distributed crawler.
- Implemented rotating proxies and user-agent spoofing.
- Stored data in a normalized PostgreSQL schema.
- Built a dashboard to monitor crawl health.

## Outcome
Provides a steady stream of fresh data for competitive analysis, enabling real-time pricing adjustments.
